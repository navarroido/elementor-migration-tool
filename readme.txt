A simple tool to help migrate your WordPress site to Elementor hosting with step-by-step instructions.

== Description ==

The Elementor Migration Tool provides a straightforward interface to guide you through migrating your WordPress site to Elementor hosting. It includes:

* Step-by-step migration instructions
* Video tutorial guidance
* Automatic installation of required migration plugins
* Simple and intuitive interface

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/elementor-migration-tool` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Elementor Migration menu item to access the tool

== Frequently Asked Questions ==

= Is this an official Elementor plugin? =

No, this is a third-party tool designed to help users migrate to Elementor hosting.

= Do I need other plugins? =

Yes, this tool requires the Migrate Guru plugin, which will be automatically installed if not present.
